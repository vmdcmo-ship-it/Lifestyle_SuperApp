# Merchant App - UI/UX Guide

> **Hướng dẫn thiết kế giao diện chi tiết cho Merchant App**

---

## 🎨 Design Principles

### Core Principles

1. **Professional & Trustworthy** - Merchant là đối tác kinh doanh
2. **Data-First** - Highlight metrics và insights
3. **Efficiency** - Minimize taps, maximize productivity
4. **Mobile-Optimized** - Touch-friendly, responsive
5. **Vietnamese-Localized** - Language, culture, UX patterns

### Design Philosophy

```
Clarity > Decoration
Function > Form
Speed > Animation
Trust > Trendy
```

## 🎨 Color Usage Guidelines

### When to Use Gold (#FDB813)

```
✅ Primary actions (Xuất bản, Xác nhận)
✅ Active states (selected tab)
✅ Important metrics (revenue, ratings)
✅ Success indicators
✅ Brand elements (logo, accents)
```

### When to Use Purple (#2E1A47)

```
✅ Text (headings, labels)
✅ Secondary buttons
✅ Icons (default state)
✅ Borders (strong emphasis)
✅ Background (premium cards)
```

### When to Use Red (#DC143C)

```
✅ Destructive actions (Xóa, Hủy đơn)
✅ Error states
✅ Urgent alerts (Đơn hàng cấp bách)
✅ Negative metrics (cancellation rate)
✅ Notification badges
```

### When to Use White/Silver

```
✅ Card backgrounds (#FFFFFF)
✅ Screen backgrounds (#F8F8F8)
✅ Dividers (#E0E0E0)
✅ Disabled states (with opacity)
✅ Text on dark backgrounds
```

## ✅ Accessibility Guidelines

### Color Contrast

**Text on Backgrounds:**
```
✅ #2E1A47 on #FFFFFF → 12.5:1 (Excellent)
✅ #4A4A4A on #FFFFFF → 9.7:1 (Excellent)
✅ #FFFFFF on #2E1A47 → 12.5:1 (Excellent)
✅ #FDB813 on #2E1A47 → 5.2:1 (Good)
⚠️ #FFFFFF on #FDB813 → 1.9:1 (Large text only)
```

### Touch Targets

```
Minimum: 44px × 44px (Apple HIG)
Comfortable: 48px × 48px
Recommended: 56px (primary actions)

Spacing between targets: 8px minimum
```

### Font Sizes

```
Minimum body text: 16px (readability)
Minimum interactive text: 14px
Caption minimum: 12px
```

## 📱 Responsive Breakpoints

### Mobile First

**Small (< 375px):**
- Single column layouts
- Smaller padding (12px)
- Compact cards

**Medium (375px - 428px):**
- Standard layouts (as shown)
- Normal padding (16px)
- Full feature set

**Large (> 428px) - Tablets:**
- 2-column grids where applicable
- Larger padding (24px)
- More whitespace

**Mobile (< 768px):**
- Same as mobile app

**Tablet (768px - 1024px):**
- 2-column product grid
- Sidebar navigation (persistent)
- Larger cards

**Desktop (> 1024px):**
- 3-column product grid
- Sidebar + main content
- Dashboard widgets side-by-side
